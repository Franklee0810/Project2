# Project2
스터디그룹 검색 사이트 

내가 한 것들 : 
1. 지역/카테고리별 스터디 검색 기능
2. 스터디 목록/상세보기
3. 스터디 리뷰 기능(Ajax) 
4. 스터디에 별점을 주면, 해당 스터디의 평점을 계산하여, 평점이 가장 높은 순으로 '인기' 스터디 정렬
